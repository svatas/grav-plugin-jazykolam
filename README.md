
# Jazykolam 1.3.3 — DEBUG in Admin + Console export

- **Admin toggle** under *Plugins → Jazykolam → Debug* (markers, mode, badge, console export).
- **`jazykolam_debug_panel()`** shows floating panel; when **console** is enabled, it also prints `console.table([...])`.
- Works in **Gantry** particles, too.

Use on demand through URL flag: `?jazykolam_debug=1`.
